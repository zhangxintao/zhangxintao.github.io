    Problem: Flip an Image

Analysis:

输入是一个只包含0和1的矩阵，代表一张图片。要求Flip -> Invert这张图片

Flip: 即把每行头尾对调

Invert: 即把所有0->1, 1->0

思路： Invert只需要遍历数组然后用if-else赋值即可，Flip可以对每行设置一个头尾指针，然后从两端向中间兑换元素

Problem: Array Partition I

Analysis:

输入是一个包含偶数个元素的数组，要求两两分组之后，取每个分组的最小值，然后对所有最小值求和，要求计算这个最小值和的最大值

以题目中[1,4,3,2]为例，最小值1无论怎么组合都一定会被取到，最大值4无论怎么组合都一定不会被取到。由于每个数组所有数相加的总和是一定的，要让最小值的和最大，就要让每个组合的两个数尽量接近，这样就能把取最小值过程中的牺牲降到最低。

思路： 对输入数组做一个排序，然后以step=2的间隙，依次取出所有的数相加，就是分组取最小值之后可能的最大和
