Squares of a sorted array

Analysis

负数经过square之后，有可能会比正数的square大，例如，

class Solution {
public int[] sortedSquares(int[] A) {
int[] result = new int[A.length];

    for(int i = 0; i < A.length; i++) {
        if (A[i] < 0) {
            A[i] = 0 - A[i];
        }
    }

    Arrays.sort(A);

    for(int i = 0; i < A.length; i++) {
        result[i] = A[i] * A[i];
    }

    return result;
}

}

class Solution {
public int[] sumEvenAfterQueries(int[] A, int[][] queries) {
int[] result = new int[queries.length];
int sum = 0;

    for(int i = 0; i < A.length; i++) {
        if (A[i] % 2 == 0) {
            sum += A[i];
        }
    }

    for(int i = 0; i < queries.length; i++) {
        int index = queries[i][1];
        int newval = A[index] + queries[i][0];
        if(A[index] % 2 == 0 && newval % 2 != 0) {
            sum -= A[index];
        } else if(A[index] % 2 == 0 && newval % 2 == 0) {
            sum += (newval - A[index]);
        } else if(A[index] % 2 != 0 && newval % 2 == 0) {
            sum += newval;
        }

        A[index] = newval;
        result[i] = sum;
    }

    return result;
}

}

Why your brain need idle time

文章的主要观点是，我们的大脑需要一些放空(mind-wandering )时间。过度忙碌的大脑会影响消化吸收有意义的知识的能力。同时，放空主要是为了帮助大脑消化合并之前学习的内容，通常只需要5到15分钟就可以。但是，不要拿这些放空时间来做一些会吸引过多注意力的事情，比如听广播、看电视或者看小说，可以做一些不太需要大脑思考的肌肉运动，比如整理衣服、洗碟子。最后，文章建议每天可以花distraction-free的15分钟散散步。

My Words

这5-15分钟，和番茄工作法的间歇时间正好是一样的，所以其实严格践行番茄工作法对于大脑就是一种非常好的间歇性放空。同时，这种放空的状态，也是冥想的主要目的之一，不排斥这时候自然产生一些负面的思绪，但是相比于用其他吸引注意力的事情占用这种放空状态，了解、认识这些思绪是更有好处的。

摘要

  • When your brain is bombarded with novel stimuli or information, she says, it can struggle to generate purposefulness and meaning.

  • I think we need to recognize that the brain’s internal train of thought can be of value in itself

  • take a 15-minute, distraction-free walk in the middle of your day
