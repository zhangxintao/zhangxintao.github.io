non decreasing array

Output

Non-decreasing Array的目标是检查是否可以通过修改最多一个元素，把输入数组变成non-decreasing array.

Analysis

public boolean checkPossibility(int[] nums) {
if (nums.length <= 1) return true;

    int invalidCount = 0;
    for(int i = 0, j = 1; i < j && j < nums.length;i++, j++) {
        if(nums[i] > nums[j]) {             
            invalidCount++;
            if (invalidCount > 1) return false;

            if (i > 0 && j < nums.length-1) {
                if (!(nums[i-1] <= nums[j] || nums[i] <= nums[j+1])) return false;
            }               
        }
    }

    return true;        
}

```

Comment

与官方给的其中一种solution一直，并且time complexity和space complexity已经达到最优的O(N),O(1)
