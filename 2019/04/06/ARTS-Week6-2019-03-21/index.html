Analysis

判断一个矩阵是否是Toeplitz =>

  1. 遍历矩阵
  2. 给定一个坐标为[row][col]的元素，位于它的对角线的下一个元素是[row+1][col+1]
    • 如果元素存在，并且不相等，则此矩阵不是Toeplitz
    • 如果元素不存在，则继续便利矩阵中的下一个元素，并重复step2
  3. 如果step2中国没有找到不相等的元素，则此矩阵为Toeplitz

Is social media the new smoking?

文章以2019年3月15号，新西兰的恐袭事件作为出发点，提出虽然social media为我们带来了巨大的价值和便利，但是用户自己的贪婪以及缺乏道德底线，也在social media上创造了一个怪物。

那么，应该对social media采取什么措施呢？作者引用了烟草的管理办法——公共教育，在包装上警告，强制的年龄限制以及限制抽烟的地点，这些措施将英国的吸烟人数从1948年的80%降低到了现在的20%，认为政府应该将social media作为一种新的疾病来管理，通过和烟草类似的public awareness以及warning来减少social media带来的伤害

Tip

overwrite return values of Jasmine spyon

用jasmine写js的unit test的时候，常常会有需要mock同一个api，返回不同值的需求。如果多次使用spyon去mock同一个api,jasmine会报错：already been spied on。

其实改变mock的返回值只需要 service.api.and.return()即可
